# Cinematic Concepts

Кіновсесвіт із власним характером — авторські концепти для кіно, серіалів та анімації. Нігілістична сатира, екзистенційна трагедія, фолк-нуар, брудний реалізм, гострі й табуйовані теми — готові фундаменти для виробництва.

Шукаю однодумців та ресурси для спільної реалізації. Якщо готові об'єднати зусилля або знаєте, як це масштабувати — пишіть.
