# Cinematic Concepts

Кіновсесвіт із власним характером — авторські концепти для кіно, серіалів та анімації з готовим виробничим фундаментом. Нігілістична сатира, екзистенційна трагедія, фолк-нуар, брудний реалізм, шекспірівський футуризм — усе готове до екранізації.

Потрібні однодумці-співучасники та ресурси для спільної реалізації. Якщо готові об'єднати зусилля або знаєте, як це масштабувати — пишіть.
