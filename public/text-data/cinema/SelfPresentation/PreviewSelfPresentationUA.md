# Проекти Самопрезентації

Нижче зібрані ідеї для швидкого старту на чистій ініціативі. Це проєкти для тих, хто готовий пірнути в невідомість заради досвіду, драйву та цікавого результату.

Якщо ви тільки плануєте у контент-мейкерство або хочете спробувати нові формати — давайте зробимо це разом. Усі реалізовані проєкти можете використовувати на своїх каналах.
