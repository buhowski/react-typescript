# Проекти Самопрезентації

Нижче — готові сценарії для старту на чистій ініціативі. Для тих, хто готовий пірнути в невідомість заради досвіду та результату.

Якщо плануєте у контент-мейкерство або шукаєте нові формати — зробимо їх разом. Усі реалізовані проєкти можете використовувати на власних каналах.
