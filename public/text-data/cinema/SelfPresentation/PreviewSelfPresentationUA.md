# Пілотні проєкти

Готові авторські сценарії та концепції відеошоу для презентації контркультурної екосистеми. Ляпас застарілому мисленню та інтелектуальна провокація: чорний гумор, естетика абсурду, гостра сатира, табуйовані теми.

Потрібні кріейтори-співучасники, щоб міняти правила гри.

Усе реалізоване можете використовувати на власних каналах.
