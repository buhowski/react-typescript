# Проекти Самопрезентації

Нижче — готові сценарії та концепція для старту — потрібна лише команда. Для тих, хто готовий пірнути в невідомість заради досвіду та результату.

Якщо плануєте у кріейтерство або шукаєте нові формати — зробимо їх разом. Усі реалізовані проєкти можете використовувати на власних каналах.
