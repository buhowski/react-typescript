# Пилотные проекты

Интеллектуальная провокация (импровизация, чёрный юмор, эстетика абсурда) как инструменты самопрезентации для создания контркультурного журнала.

Сценарии и концепции готовы к старту — нужна лишь команда.

Ищу криэйторов, чтобы менять правила игры. Все реализованные проекты сможете использовать на собственных каналах.
