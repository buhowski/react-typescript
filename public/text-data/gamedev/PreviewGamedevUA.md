# Gamedev Concepts

Цифрові виміри із власним характером — авторські концепти відеоігор та складного інтерактиву з готовим виробничим фундаментом. Наративні лабіринти та інтелектуальна відвертість як основа для створення нових віртуальних світів.

Потрібні однодумці-співучасники та ресурси для спільної реалізації. Якщо готові об'єднати зусилля або знаєте, як це масштабувати — пишіть.
